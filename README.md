This project builds a machine learning model that predicts NFL wide receiver production (yards, receptions, touchdowns) using college performance and combine metrics.

Features:

Gradient Boosting Regression (scikit‑learn)
Full preprocessing pipeline (cleaning, imputation, scaling)
Three independent regression models for yards, receptions, and TDs
Interactive user input for new prospect evaluation
Data validation and feature engineering

Tech Stack:

Python
Pandas
NumPy
scikit‑learn
Jupyter Notebook

Purpose
This project was created to demonstrate practical machine learning skills for football analytics, including model development, evaluation, and real‑world application to player projection. 
I was not able to grab a super comprehensive dataset with the time I had, so this program was not neccesarily made to be very accurate, but to just show you that I have the skills for the job.
